# catchMackerel

Makes use of the catchR package to
 * read in all mackerel data from the commercial fleet (catch, length-frequencies, bio data)
 * analyse this catch, overall and by region
 		- total
 		- catch-at-age
 		- weight-at-age in the catch
 		- length frequencies
